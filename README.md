# TakeYourTicket

A cinema ticket selling API

## Endpoints:

### Movies

POST - create movie

PUT - update movie

### Session

GET - Find a session by movie name and day

POST - create session

PUT - update session

### Sale

POST - record a ticket selling

### It must

--> Record logs of the operations

--> Use SQLServer as database (verify the attatchment)

--> OPTIONAL: use OAuth
